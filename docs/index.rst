Welcome to ``cryptography``
===========================

``cryptography-gocrypto`` is a backend for the `cryptography`_ Python library.
It provides an interface to use the standard cryptography API with Go's crypto.

**At this time this should be considered experimental software and not ready
for any sort of production use.**

Installation
------------
git clone the `cryptography-gocrypto`_ repository from GitHub, ``cd`` into
the directory, and then run ``pip install .``.
